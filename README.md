##webdemo是什么?

示例http server 与 client 通信。

目前有两个Demo,demo1\demo2

demo1诠释了httpserver与httpclient，完成了Get和Post方法示例，其中Post又分
Post和PostForm。Post中包括两种方式传送表单数据，一个通过url.values、一个通过
JSON传递

demo2是从网上摘抄的代码，比较简单清晰，放在这里供需要的人参考。
